# 🚀 Vyapix

Vyapix is a full-stack MERN (MongoDB, Express, React, Node.js) application designed to manage and streamline inventory and sales operations efficiently. Built as a college project, it demonstrates real-world backend architecture, authentication, and scalable structure.

---

## ✨ Features

* 🔐 User Authentication (JWT-based)
* 📦 Inventory Management
* 💰 Sales Tracking & Reports
* 📊 Backend API with structured controllers & routes
* ⚙️ Scalable MERN architecture

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/Abdulrahman-programmer/Vyapix.git
cd Vyapix
```

### 2. Setup Backend

```
cd backend
npm install
```

### 3. Create Environment File

Create a `.env` file inside backend:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### 4. Run Backend

```
npm start
```

---

## 🔐 Environment Variables

| Variable   | Description                   |
| ---------- | ----------------------------- |
| PORT       | Server port                   |
| MONGO_URI  | MongoDB connection string     |
| JWT_SECRET | Secret key for authentication |
